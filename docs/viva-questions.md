# Viva Questions and Answers

## 1. What is image classification?
**Answer:** It is assigning an image to a predefined category such as healthy leaf or tomato blight.

## 2. What is CNN?
**Answer:** A Convolutional Neural Network learns visual patterns using filters and layers.

## 3. Why CNN is used for plant disease detection?
**Answer:** Leaf diseases are visible through color, texture, spots, and shape patterns.

## 4. What is confidence score?
**Answer:** It is the model estimated probability for the predicted class.

## 5. What is model training?
**Answer:** Training adjusts model weights using labeled images.

## 6. What is dataset preprocessing?
**Answer:** It includes resizing, normalization, augmentation, and train-test splitting.

## 7. What is FastAPI?
**Answer:** FastAPI is a Python web framework for building high-performance APIs.

## 8. Why React is used?
**Answer:** React provides reusable UI components and interactive dashboards.

## 9. How does upload flow work?
**Answer:** The frontend sends multipart image data to FastAPI, which validates and stores the image.

## 10. How does AI assistant help?
**Answer:** It explains disease results, treatment, prevention, architecture, and viva topics.

## 11. What are future enhancements?
**Answer:** Real model, mobile app, multilingual support, weather alerts, IoT, and expert module.

## 12. What is severity level?
**Answer:** It indicates practical impact such as Low, Medium, High, or Critical.

## 13. What is prediction history?
**Answer:** It stores previous analyses for review and reports.

## 14. What is disease knowledge base?
**Answer:** It is curated disease information used by UI and future RAG workflows.

## 15. What is SQLite?
**Answer:** A lightweight file-based database for local development.

## 16. How is PostgreSQL supported?
**Answer:** SQLAlchemy uses DATABASE_URL, so the database can be switched.

## 17. What is JWT?
**Answer:** A signed token used for stateless authentication.

## 18. Why mock ML first?
**Answer:** It lets the full app work before a trained model is available.

## 19. What is OpenRouter?
**Answer:** An API gateway for LLM models used by the assistant.

## 20. Why keep API keys backend-side?
**Answer:** To prevent exposing secrets in browser code.

## 21. What is CORS?
**Answer:** A browser security mechanism controlling cross-origin API requests.

## 22. What is multipart/form-data?
**Answer:** A request format for file uploads.

## 23. How are images validated?
**Answer:** By MIME type and size limit.

## 24. What is RAG-ready design?
**Answer:** Knowledge files can later be retrieved and inserted into AI prompts.

## 25. What is Recharts used for?
**Answer:** Analytics charts in dashboards.

## 26. What is GSAP used for?
**Answer:** Landing page animations only.

## 27. What is Tailwind CSS?
**Answer:** A utility-first CSS framework for rapid UI styling.

## 28. How are reports generated?
**Answer:** A print-friendly HTML page supports browser print/save as PDF.

## 29. What is SQLAlchemy?
**Answer:** Python ORM for mapping database tables to classes.

## 30. What is Pydantic?
**Answer:** Validation library for request and response schemas.

## 31. What is the disclaimer?
**Answer:** Outputs are advisory and educational, not guaranteed diagnosis.

## 32. How can accuracy improve?
**Answer:** Train on more diverse labeled leaf images and validate with experts.
